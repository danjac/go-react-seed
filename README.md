React-go seed
===================

Basic set up for starting a ReactJS and Go application.

It's built with the following tech:

- ES6 + Babel
- React
- React-router
- Alt
- Go 
- Gin-gonic
- Twitter bootstrap

Setup
-----

1. **npm install**
1. **gulp**
1. **go run main.go**
1. Open a browser at *localhost:3000* and you should be good to go.
